# 柯布教育网站

## 背景
柯布生态学院是一个国际教育组织，专注于推广其教育理念，提倡教育的目的不应只是服务人类，更要融合生态理念，实现人类的可持续发展。

从黄山打完黑客松回来以后，朋友推荐的一个活儿，因为是类似公益网站，没啥钱并且需要降低维护成本，本来打算用express的但是考虑到后面做品牌升级，加上还有个管理后台，可能要定制。

所以最后经过调研，用react框架结合supabase数据库实现了整个网站。然后通过vercel进行部署，实现了0成本部署，后续假设用户数量大迁移加钱也方便，并且也不算贵。

网址：[柯布生态学院](https://www.cobbecociv.academy/)

## 复盘
1. 项目从7月20多号开始拉群沟通，但是现在三个月了还没有结束，主要有三个原因：
  * 需求不明确，几乎一半时间都用来沟通网站真实需求了
  * 我个人时间有限，没办法一直持续沟通
  * 与甲方有时差，加上她也比较忙，每次对进度都要预约，比较麻烦

2. 项目在启动前商议并签署项目协议很有必要
  
  * 商议了一个项目包括项目`需求总结`、`技术选型`、`分阶段付款`和`各阶段计划交付时间`等，并且注明了因为**甲方因素导致的延期可以协商处理，不扣钱**。
  
  * 当下因为甲方那边付款资质问题导致网站支付功能受阻，所以我这边只能等，不过不用担责，还行。并且也拿到了2笔阶段款

3. UI风格
  
  * 找了打比赛认识的一个设计师，我自己掏了3000出来，但是用户似乎更喜欢传统的express的那种风格 :cry:

  * 此外对于年纪大一点的用户，后台管理用所见即所得的方式似乎更棒，用户理解起来也更方便。

4. supabase 免费的版本有个缺点就是没过一段时间没人访问就会暂停数据库资源，其实这是好事，节省资源，只是正式上线可能需要升级到付费版本
