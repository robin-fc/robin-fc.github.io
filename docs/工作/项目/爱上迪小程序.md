# 爱上迪小程序

## 背景
爱上迪小程序是一个为了做迪士尼周边二手市场而开发的小程序，属于千岛这类产品，只是垂类。这个活儿是朋友的朋友的推荐的。合作的是一个有多年经验的产品，但是失业了，想搞自己的事情，时间催的很紧。

3. UI风格
